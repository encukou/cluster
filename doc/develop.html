Clustering visualization – Development

Prerequisites

To work on the Clustering GUI, you will need the Qt SDK and Bazaar.

For Windows, download Qt SDK (direct download link), and Bazaar (choose the latest Standalone stable release).

For Linux, just install the appropriate packages (sudo apt-get install libqt4-dev bzr should work in Debian/Ubuntu).

Working with Bazaar

Get the source with the bzr branch http://cs.joensuu.fi/~pviktor/cluster command (or Branch that address from the Bazaar Explorer GUI). This will create a “cluster” directory with your personal copy of the source.

Once you make significant changes, use the bzr ci command (or Commit from the GUI). This will save your changes locally.

When you are ready to publish your changes, copy the “.bzr” directory to some accessible place (like cs.joensuu.fi/~yourlogin/cluster/.bzr) and send an e-mail to pviktor@cs.joensuu.fi about it.
(unfortunately, the SFTP server at cs.joensuu.fi doesn't seem to work with Bazaar, so you can't use bzr push easily).

Compiling

Run qmake && make to compile the code on Linux.

On Windows, use the big green Run button in Qt creator (which you installed along with Qt).
Alternatively, put bzr, qmake and mingw32-make in your %PATH% and run the windist.bat file included in the source.

Generating documentation

If you want to create the automatically-generated documentation, you will need to install Doxygen, and download the qt.tag file into the main source directory. After that just run the doxygen command.
