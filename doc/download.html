Windows

The program, with all required libraries, is here. Just unzip it somewhere and run cluster.exe.

If you are upgrading from a previous version, you don't need the libraries. In this case download only cluster.exe, and replace the previous one.

Linux & other systems

Currently you will need to build from source. Install Qt (libqt4-dev in Debian/Ubuntu), then get the release tarball and run qmake && make.

See the development page if you want current source

Data sets

Some example data sets are provided at the SIPU dataset page.
